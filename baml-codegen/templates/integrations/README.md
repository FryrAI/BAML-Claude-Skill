# Integration Templates

Framework-specific integration code templates

## Supported Frameworks

### Python
- FastAPI
- Flask
- Django

### TypeScript
- Next.js
- Express
- NestJS

### Ruby
- Rails
- Sinatra

### Java
- Spring Boot

### Go
- Gin
- Echo

### C#
- ASP.NET Core

## Template Structure

Each integration template includes:
- API endpoint/route
- Request/response models
- Error handling
- Async patterns
- Configuration examples
